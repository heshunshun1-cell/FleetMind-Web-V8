FleetMind

AI Logistics Assistant

FleetMind is a small logistics analysis web app inspired by my experience with freight operations. I built it to help understand truck revenue, operating costs, profit margin, risk level, and the main cost pressure in a simple and clear way.

View Fleet Dashboard View Fleet Analytics View Operational Insights View Route Analytics View Truck Records View Trip Records Add New Trip Ask RAG Assistant

Project Information

This project is based on realistic logistics scenarios, but all sample data has been anonymised and modified for privacy protection. It does not include real company names, driver names, plate numbers, clients, or private routes.
